# Scripts

- `uie_closure_guard.py`: closure guard for this repository.
- `extract_constants.py`: evaluates explicit formulas from `artifacts/constants_extraction_inputs.json`.
- `promote_constants.py`: promotes extracted constants into the registry and stitch files.
- `update_manifest.py`: refreshes the SHA256 manifest.
- `release_gate.py`: checks manifest integrity and fully-extracted release status.
